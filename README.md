# Arduino 摩斯电码 LED 项目说明

## 硬件连接
- **默认输出引脚**：`13`
- **连接方式**：将LED正极连接到Arduino引脚13，负极连接到GND

## 软件配置

### 方法一：使用本软件
1. 在本软件的**右侧结果栏**获取生成的代码
2. 复制所有代码
3. 打开Arduino IDE
4. **完全替换**现有代码
5. 上传到Arduino开发板

### 方法二：直接打开文件
- 在**本软件的高级目录**下找到 `morse2led.txt` 文件
- 直接打开并复制其中的代码

##  常见问题
- **LED不亮**：检查引脚连接和LED极性
- **代码上传失败**：检查开发板选择和端口设置
- **闪烁模式错误**：确认复制了完整代码

---

如需调整闪烁速度或修改输出引脚，可在代码中找到相应参数进行修改。
